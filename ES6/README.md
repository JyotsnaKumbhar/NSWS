# ES6+ Refactor Project – User Manager

This project demonstrates how to refactor ES5-style JavaScript code into modern ES6+.

## Concepts Used
- let / const
- Arrow functions
- Template literals
- Classes
- Destructuring
- Spread operator
- Modules
- Async / await

## Run
node src/app.js

## Files
- userManager-es5.js → Old ES5 version
- userManager-es6.js → Refactored ES6 version
- data.js → Sample dataset
- app.js → Main file with async example
